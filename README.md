# AIML-Task-4-Classification-with-Logistic-Regression.
Build a binary classifier using logistic regression. 

The objective of this project is to build a binary classification model using Logistic Regression to predict whether a breast tumor is malignant (cancerous) or benign (non-cancerous). The project demonstrates the complete machine learning workflow including data preprocessing, feature scaling, model training, evaluation, threshold tuning, and probability interpretation using the sigmoid function.

The implementation process involved several important machine learning steps. First, the dataset was divided into training and testing subsets to evaluate the model on unseen data. Since Logistic Regression is sensitive to feature scales, StandardScaler was applied to standardize all input features and improve model performance. The Logistic Regression classifier was then trained using the processed training data. After training, predictions and probability scores were generated for the testing dataset. Multiple evaluation techniques were used to analyze performance, including Accuracy Score, Confusion Matrix, Precision, Recall, F1-Score, ROC Curve, and ROC-AUC Score. A confusion matrix was visualized to examine correctly and incorrectly classified instances, while the ROC curve was plotted to assess the model's ability to distinguish between malignant and benign cases. Additionally, threshold tuning was performed to observe how changing the classification threshold affects prediction outcomes. The sigmoid function, which is the core mathematical component of Logistic Regression, was also visualized to understand how the model converts linear outputs into probabilities ranging from 0 to 1.

The results demonstrated that Logistic Regression is highly effective for breast cancer classification. The model achieved very high accuracy, typically exceeding 95%, with strong precision and recall values, indicating that it successfully identified malignant and benign tumors with minimal classification errors. The ROC-AUC score was close to 1, showing excellent class separation and strong predictive capability. Analysis of the confusion matrix revealed very few false positives and false negatives, which is especially important in medical applications where incorrect diagnoses can have serious consequences. Threshold tuning further highlighted the trade-off between sensitivity and specificity, allowing the classifier to be adjusted according to different application requirements. Overall, the project successfully showcased the practical implementation of Logistic Regression, demonstrated the importance of preprocessing and feature scaling, and provided valuable insights into classification evaluation techniques. The work strengthened skills in data preprocessing, supervised machine learning, statistical evaluation, model interpretation, probability-based prediction, and healthcare data analysis.
